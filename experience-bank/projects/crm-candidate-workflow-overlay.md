# CRM Candidate Workflow Overlay

Improved a software tool that reduced friction when recruiters interacted with the CRM. The tool acted like a CRM overlay with drag-and-drop candidate movement between recruitment stages.

## What It Did

- Reduced friction in recruiter CRM workflows.
- Provided drag-and-drop interactions for moving candidates between stages.
- Supported internal recruitment operations as a production tool.

## CV Bullet Raw Material

- Improved an internal CRM workflow overlay that let recruiters move candidates between stages with drag-and-drop interactions.
