# InBetments Football Analytics and Prediction Model

Built a football analytics and prediction side project using XGBoost and historical football data from API-Football. The project used a large historical dataset, roughly one million football matches, and achieved around 53% cross-league accuracy on three-way match result prediction: home win, draw, or away win.

## What It Did

- Used API-Football historical match data.
- Built a football prediction model using XGBoost.
- Worked with roughly one million football-match records.
- Evaluated three-way match result predictions across leagues, reaching about 53% accuracy.
